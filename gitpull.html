O que é um Git Pull

Recurso utilizado para acessar e copiar os arquivos em um repositório remoto, este comando irá realizar um clone da branch selecionada, com isso você poderá realizar as alterações necessárias e encaminhar ao GITHUB, por exemplo), os arquivos atualizados, em forma de commit.
