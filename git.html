O que é GIT

O GIT é um sistema de controle de versão distribuído criado com o intuito de facilitar o gerenciamento de versões do código, que ganhou espaço entre os desenvolvedores, pelo fato do próprio sistema trabalhar com o backup das informações de maneira mais completa, a partir dele conseguimos controlar as mudanças, saber quem fez, recuperar versões anteriores, testar novos projetos sem interferir no código principal, entre muitas outras tarefas comuns ao dia a dia de qualquer projeto, mas que sem o GIT poderiam trazer problemas e dificuldades.
