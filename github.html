O que é GITHUB

É uma plataforma de hospedagem de código fonte que utiliza o Sistema de Controle de Versão GIT. Ela permite que desenvolvedores se auxiliem em projetos privados e/ou Open Source (Código Aberto) em qualquer lugar do mundo.
